使用css实现一个持续动画效果

发表于 2018-04-13

上面css中采用简洁写法，如下为动画的主要参数

参数 描述
animation-name 规定需要绑定到选择器的keyframe名称
animation-duration 规定完成动画所花费的时间，以毫秒计
animation-timing-function 规定动画运动速度曲线
animation-delay 规定动画在延迟多少秒后开始运动
animation-iteration-count 规定动画播放的次数
animation-direction 规定动画是否轮流反向播放

具体细节请参照http://www.w3school.com.cn/css3/css3_animation.asp
